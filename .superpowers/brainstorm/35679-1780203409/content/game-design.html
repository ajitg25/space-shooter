Space Shooter — Game Design

Browser-based, HTML5 Canvas, pixel/retro style

Architecture
File Structure
📁 interactive-game/
📄 index.html    — shell + canvas element
📄 game.js       — game loop, entities, input
📄 sprites.js    — pixel art drawn via Canvas API
📄 styles.css    — dark background, HUD font
Game Loop
States & Flow
MENU → PLAYING → WAVE CLEAR → BOSS → PLAYING
                        ↓
                GAME OVER → MENU
Entities
Player Ship
• Arrow keys / WASD to move
• Spacebar to shoot
• 3 lives, blinks on hit
• Fires single shot (upgrades to spread shot at wave 3)
Enemies
• 3 types: drone, fighter, kamikaze
• Formation patterns per wave
• Shoot back at player
• Boss every 5 waves (multi-phase HP)
Pickups
• Shield restore
• Spread shot (temp)
• Extra life (rare)
• Score multiplier
Visuals
• Scrolling star parallax bg
• Pixel sprites (Canvas drawImage)
• Explosion particle effects
• Retro pixel font (Press Start 2P)
Scoring
Drone: 100pts  |  Fighter: 250pts  |  Kamikaze: 150pts  |  Boss: 1000pts
Wave clear bonus: wave × 500pts  |  Multiplier pickup: 2× for 10 seconds
Waves
Wave 1–4: drones only, simple formations
Wave 5: Boss #1 — slow, 3 phases, shoots spread
Wave 6–9: drones + fighters, more bullets
Wave 10: Boss #2 — faster, homing missiles
Wave 11+: all enemy types, random formations
